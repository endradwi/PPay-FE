## Getting Started

This project worker can follow the steps below:

### Prerequisites

* install node.js

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/endradwi/PPay-FE.git
   ```
2. Run nodejs
   ```sh
   npm install
   ```
3. please configure in .env

<p align="right">(<a href="#readme-top">back to top</a>)</p>



<!-- USAGE EXAMPLES -->
## Usage

1. Run
   ```sh
   npm run dev
   ```
2. Run application at browser
   ```sh
   http://localhost:5173
   ```
